A C++ console program that helps you track and understand your thinking patterns. Uses a Binary Search Tree to store the frequency of common cognitive distortions.  

 Features:
- Log how you feel today 
- Track patterns over time 
- See Top 3 distortions with explanations 
- Get advice for your #1 distortion 
- Beginner-friendly & interactive 

 Distortions Included:
- Catastrophizing: Expecting the worst  
- Mind Reading: Assuming what others think  
- All or Nothing Thinking: Seeing extremes only  
- Overgeneralization: Turning one event into a pattern  
- Emotional Reasoning: Believing feelings = facts
